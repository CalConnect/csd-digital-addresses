=== AddressClassDescription
An AddressClassDescription can be specified in the following object structure (in JSON)

[source,json]
----
addressClasses: {
  streetAddress: {
    description: Street Address,
    availableFields: [ (addressComponent) 1..* ... ]
displayTemplate: { ... ],
inputTemplte: {}

  }
}
----
